Приложение создает веб браузерное окно с возможностью запуска js кода в отдельном потоке. Реализованы методы для отправки сообщений в youtube live_chat, vk в личные сообщения.
Решение платформозависимое(подходит только под windows).
Использовалось Qt, winapi(для имитации действий пользователя - нажатия клавиш клавиатуры).
